# 音频文件说明

把你的音频文件放进这个文件夹，游戏会自动启用它们。
**没有这些文件游戏也能正常运行**（音频系统会优雅降级为静音）。

## 音效文件（短音效，建议 .mp3，每个 < 100KB）

| 文件名 | 触发时机 |
|--------|---------|
| `card.mp3` | 打出普通卡牌 |
| `attack.mp3` | 打出攻击/禁忌卡 |
| `shield.mp3` | 打出防御卡 |
| `hurt.mp3` | 玩家受伤（预留） |
| `heal.mp3` | 回复生命/理智（预留） |
| `awaken.mp3` | 触发觉醒 |
| `victory.mp3` | 战斗/通关胜利 |
| `defeat.mp3` | 死亡失败 |
| `select.mp3` | 选择角色/事件选项 |

## 背景音乐（循环播放，建议 .mp3）

| 文件名 | 播放场景 |
|--------|---------|
| `bgm_map.mp3` | 地图/角色选择界面 |
| `bgm_battle.mp3` | 普通战斗 |
| `bgm_awaken.mp3` | 觉醒状态下的战斗/地图 |

## 去哪里找免费音频？

- **freesound.org** — 海量 CC 授权音效（注意看授权条款）
- **opengameart.org** — 专为游戏制作的免费素材
- **incompetech.com** — Kevin MacLeod 的免费配乐（标注署名即可商用）
- **pixabay.com/sound-effects** — 免版税音效

下载后改成上面对应的文件名放进来即可。文件名必须完全一致。
